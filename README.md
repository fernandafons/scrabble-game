# Scrabble game

## getseven.js
This is a simple program that returns a list of 7 random letters from the alphabet and of which 3 for sure will be vowels.

## getseven.js
This program returns one random letter from the alphabet.

## running the project
To run this program you need to have node installed.
After clonning this repository, in the project directory, run `node [name-of-the-file-you-want-to-run]`
